# Phicomm N1

本仓库编译的 N1 固件大致上分为 LEDE，iStoreOS 的直接编译和 ImmortalWrt，penWrt 的 Armbian 内核打包两种类型，在刷机和初始化配置上会有些不同。而且每个固件由于自身代码以及一些组件的版本不同所以包含的插件也会有一定的差异。下面会详细的列出这些差异以及固件的基本刷机、升级以及初始化配置方法。
